For proper operation with PC card these components should be installed / configured:
/etc/pcmcia/fixed.opts
uncomment DO_MOUNT and MOUNTPT add 
RUN_COMMAND="y"
USER="pia"
COMMAND="~pia/pia/bin/copy_j6i $MOUNTPT ~pia/.pia/Photo_Album"
/etc/pcmcia/fixed
add 
if [ ! $FAIL -a "y$RUN_COMMAND" = "yy" ] ; then
    if [ "x$USER" = x ] ; then
        $COMMAND || FAIL=1
    else
        su $USER -c "$COMMAND" || FAIL=1

    fi
fi
~pia/pia/bin/copy_j6i
should exist and be suid pia
