Masters

This directory contains a directory of master files for each template or form collection. By convention, subdirectories with uppercase names contain complete templates; subdirectories with lowercase names contain collections of forms and prototypes that are shared among many templates.

Each template directory contains:

template.inc
a <dl> of name-value pairs that control the agent
data.inc
default values for form.inc; a new data.inc file is constructed in the Agent's data directory to contain the values the user has supplied.
Further information can be obtained from the Implementation Notes.
Note:
Browsing below this level under /Template (i.e. the Template agent, as opposed to a DOFS) may result in your trying to execute master files as InterForms, with unpredictable results.
