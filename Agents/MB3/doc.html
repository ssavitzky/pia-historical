Documentation for MB3

This agent is used for scheduling and controlling the Ricoh MB-3 Media Board. See configuration files for instructions on how to download and run a local version.

Notes on the MB3 device

Touchscreen
The touch screen currently works. Tapping emulates a mouse click (accompanied by a beep).
Mouse
A normal mouse attached to the computer (open the top drawer).
Screen
The screen operates at 640x480 resolution.
DVD
The DVD drive will read digital video disks or normal CD's. It will NOT read CD-R disks (CD-RW disks are OK.)

Notes on the MB3 agent / PIA

Proxy
Browsers on the MB3 device should have the PIA &piaHOST;:&piaPORT; set as their proxy. This allows the history agent to keep track of all Web pages visited during a presentation, and the CIA agent to add navigation bars to the visited pages.
Events
Events are stored in a file corresponding to the day of the event. These files are located in &usrDIR;/&agentName;/YYYY/MM/DD (YYYY=year, MM=month, DD=day).
Annotations
The plan is to have screen annotations (drawings) to be automatically saved to to a directory. These annotations would then be converted to gif or jpg files and viewable as part of the presentation.
