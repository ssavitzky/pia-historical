.../pia/Agents

This directory contains a directory for each agent or agent type; each agent or type is contained in a subdirectory. This allows each agent to consist of multiple CGI scripts, documents, and InterForms, and still be addressable by a single, simple URL.

This directory is ``overlaid'' by your personal Agents directory, /~/Agents/; this lets you make private versions of any agent's forms.
