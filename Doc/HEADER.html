[..] [files]

This directory contains documentation generated by members of the PIA Group at CRC. A new user or visitor should start with a description of the PIA group, an Introduction to PIA's, or the PIA vision. Experienced users may want to head straight to the documentation overview.

Any document that is not a self-contained HTML file will normally have its own subdirectory; that way it can have whatever structure it needs and still be addressable as a single, simple URL.
