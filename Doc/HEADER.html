This directory contains documentation generated by members of the PIA Project at CRC. A new user or visitor should start with An Introduction to PIA's.

Any document that is not a self-contained HTML file should have its own subdirectory; that way it can have whatever structure it needs and still be addressable as a single, simple URL.
