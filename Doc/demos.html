PIA-Based Appliance Demos

A working prototype has been developed and is available for "playing". At any given time, running PIA-based appliances can be found at:
  • PhotoAlbum
  • CD-aRchiver
  • The WebWort / WebPrinter
and personal Agencies can be found at:
  • The PIA group's agency
  • Steve's agency
  • Greg's agency
The bravest among you might even like to run your own agencies and develop your own agents. This is straightforward for unix users, simply run: 
  /pia1/pia/bin/pia
and browse to the URL it prints out. If you are on a shared server and a PIA is already running there, you may have to supply a different port number; see the User's Manual for details.

To develop agents, you will need a working copy in your home directory: 

  setenv CVS_ROOT /pia1/CvsRoot
  cd
  cvs checkout pia
