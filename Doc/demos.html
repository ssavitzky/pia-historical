PIA demos

A working prototype has been developed and is available for "playing". At any given time, running agencies can be found at:
  • The WebWort / WebPrinter
  • Steve's agency
  • Greg's agency
The bravest among you might even like to run your own agencies and develop your own agents. This is straightforward for unix users, simply set your CVS_ROOT environment variable to /pia1/pia and do a cvs checkout pia, then run pia/src/Agencies/small_perl/pia.pl and away you go... (also see the manuals).
