This document describes how to create Agents in the PIA framework. Agents provide some value added service and consist mainly of web pages written with an extended set of "active" tags. This document includes a tutorial, authoring manual and style guide, explanation of the tools, and a reference section for the extended tagset. About this Document Tutorial Agent Authoring Manual InterForm Manual Occasionally, the developer may need to modify the software code which comprises an agent. In such cases, the developer should start by sub-classing the GenericAgent class in piaDIR/src/java/crc/pia/GenericAgent.java Internals manual.
