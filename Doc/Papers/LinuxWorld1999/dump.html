PIA Platform for Information Applications
 
Infoware Layer
Agents: Most of an application consists of ``active documents,'' which are easy for users to maintain.  
Software Layer
Agency: Web server and Document processor  

Aspects of Infoware

  • Information is Everything
    • more important than either interface or processing
  • Information is informal:
    • mostly in documents, not databases
  • Information changes:
    • No proprietary formats or applications
  • Processing is pervasive:
    • The user must be able to do processing, too!
    • Freedom from servers and programmers

Document-Oriented Computing

Infoware needs...

  • A whole new way of looking at computation
    • Computing with documents not data
  • A new location for computation
    • not just clients and servers
    • peers and proxies as well
    • personal and workgroup processing

We call it Document-Oriented Computing
