Open Source Goals
  1. Enhance Ricoh's ability to build information applications
    • engage open source developers to improve core technology
    • increase speed of development, and lower cost of maintenance
    • help establish an open standard for building information applications
  2. Enable 3rd party support of Ricoh products and position Ricoh as a preferred supplier for solution providers
    • Catalyze community of developers creating information application solutions for the workgroup

        Want to Attract Solution providers

        Most Information applications will require customization.
        • VAR's/Integrators/Consultants provide solutions to the workgroup
          • They have close relationship with customers
          • They need low-cost, customizable development platform
            ... to profitably develop solutions

        First we need Initial Developer Community

        VAR's unlikely to be interested until technology is relatively mature.
        • Initial open source developers use for:
          • Information applications for their workgroups
          • personal productivity
        • Help improve technology

        How the Developer communities overlap

        Ricoh's role

        • Provide technical leadership for initial developers
          • oversee (moderate) technology development
          • communications infrastructure (web site, mailing list, etc.)
          • code repository
        • Support solution developers
          • Champion of "open standards" for customer's benefit
          • Supply horizontal appliances
          • Supply 3rd parties with customizable platforms

        Summary of Open Source effort

        Open Standards for Information application development are critical to Ricoh's ability to compete in the marketplace for next generation office machines.
        • Open Source developer community assures long-term viability of technology
        • Customization by 3rd party solution providers required to provide effective information applications
        • Open Source is best way of reaching these developers
          and improving the technology
