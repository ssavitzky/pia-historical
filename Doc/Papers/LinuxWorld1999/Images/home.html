Welcome to the PIA

This is the home page of your personal PIA. It is running as a process under your own user name, and is accessible through the URL <http://gelion.crc.ricoh.com:8888/>. Of course, if you're looking at this page in a web browser, you already know that.

You can find out more about this particular PIA server from the Agency agent, which is used to perform server management functions. You can find out about its implementation from the API (Application Programming Interface) Manual, and by directly studying the source code, all of which is online.

You can find out more about the PIA framework and its role in information appliances from the online documentation described in the table below.
