Document-Oriented Computing: A Manifesto

 

Document-Oriented a Manifesto
Computing  
  The Web is the center of modern computing. It is made up entirely of documents, some of which are active (i.e. specify computations).  
  Computing should operate on documents, and should be specified by documents.  
  Documents should be specified in a standardized markup language. Embedded computation should be specified in the same language.
