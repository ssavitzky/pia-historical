About this document

This document is one of the series of technical reports describing the PIA framework. These technical reports normally correspond to a major release of the Information Agency and are generated from the online documentation. The most up-to-date version of this information can always be found in the Doc directory of the PIA framework electronic distribution which can be obtained from the PIA group at the California Research Center of Ricoh Silicon Valley (pia@rsv.ricoh.com)

The series includes "Introduction to the PIA Framework" which provides an overview and motivation for this work, "Using the Information Agency" which shows how to install and use the Information Agency -- the core component of the framework, and "Developing Agents using the PIA Framework" which provides a tutorial, style guidelines, InterForm manual, etc. for those who wish to create new services. A glossary describing these terms can be found in the appendix of this report. Related documents on how to use the framework to build specific applications (Network Office Appliances), trends in network based computing, and other topics may be found in the electronic distribution or by contacting the PIA group.
