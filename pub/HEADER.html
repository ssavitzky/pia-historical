This is the top-level directory for the published software and documentation generated by members of the PIA Project at CRC. It is roughly equivalent to an eventual web site for the PIA.

If you check out a working copy of the pia module, you will get a copy of this directory into which your makefiles will install things (using the command make public) after you have worked on them.

Note:
People browsing this directory via the Web will probably not see this file, but index.html instead.
