Agent installation page

If you are running a PIA, you should be able to create and run agents simply by clicking on the submit buttons below. Note that currently we assume that the agents interforms have been installed in the appropriate root directories ( ../../Agents/[agent] relative to the PIA root directory, e.g. ~/pia/src/Agencies/perl_small ) You should change the root entries below before submitting if the interforms are stored elsewhere on your system. Eventually you will be able to retrieve interforms from subdirectories of this directory.
