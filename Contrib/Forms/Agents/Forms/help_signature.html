Signing The upgraded forms system uses your private key to create a digital signature of an html document -- normally a filled out form. To sign a document you must provide your "pass-phrase", however anyone can verify a signature on a given document using your public key.

The forms digital signature first creates a hash of the form to be signed, adds the user name, date, and form url then hashes and signs that data using the private key (which can only be accessed by providing the pass-phrase).
