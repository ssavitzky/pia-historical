Sometimes a user should be notified only when some form entry meets a condition. To specify this type of a route entry, choose onlyif as the "when" for an entry then (after submitting) you will be given a chance to enter a condition. An example might be to route to a user only if a variable named "var1" was greater than $1000 -- that condition would be specified <test positive><difference> &var1; 1000 </difference><test>

Otherwise specify "always" under when to always route to a user or "never" to effectively remove a user from a route.
