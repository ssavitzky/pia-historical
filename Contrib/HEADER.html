.../pia/Contrib

This directory is for software, especially configuration files, contributed by PIA users and developers.

The contents of this directory are not an official part of the PIA distribution, but are included in the hope that they will make it more useful.

Each directory in this subtree corresponds to a site, and each directory under that to a user. Under each user, the ``my/'' subdirectory contains the contents of the that user's PIA state directory (``~/.pia/'' on Unix systems) -- the name is changed to avoid having special characters in the names. Other configuration files, if the user chooses to put them here, are not renamed.
