.../pia

This is the top-level directory for software and documentation generated by members of the PIA Project at CRC.

The top-level directory /pia1/pia is a CVS-controlled working directory. It is mainly intended for work stable enough to be seen outside the Project; consequently it may not contain the latest-and-greatest version of anything, but at least it will probably work.

Annotated Contents

Agents/
Agent InterForms. Agents used primarily for software development are in src/Agents/.
Config/
Configuration files for various PIA-based appliances.
Contrib/
Contributed software.
Doc/
``pure'' (non-active) documentation, including working notes and other internal documents, which are for use inside CRC ONLY! Eventually a filtered version will be available outside CRC.
bin/
Binaries (executable files). In addition to agencies (most of which will be scripts that set environment variables before invoking an interpretor), there will be some additional applications here for use on WebWorts and other PIA-based appliances.
lib/
Libraries. In a development environment, the subdirectories may be links into src/lib/.
pub/
``Public'' directory. This is a prototype for the PIA Website; it is where you look if you want user documentation or installation packages. (Of course, it's mostly empty at the moment.) It is expected to consist mainly of links to parts of the internal tree.
src/
All source code developed locally by the PIA group, including Agencies, Agents, and ``pure'' source code meant to be installed in places like /usr/local/bin and /usr/local/lib.
