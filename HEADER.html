.../pia

This is the top-level directory for software and documentation generated by members of the PIA Project at CRC.

The top-level directory /pia1/pia is a CVS-controlled working directory. It is mainly intended for work stable enough to be seen outside the Project; consequently it may not contain the latest-and-greatest version of anything, but at least it will probably work.

Annotated Contents

Doc
``pure'' (non-active) documentation, including working notes and other internal documents. For use inside CRC ONLY!
pub
``Public'' directory. This is a prototype for the PIA Website; it is where you look if you want user documentation or installation packages.
src
All source code developed locally by the PIA group, including Agencies, Agents, and ``pure'' source code meant to be installed in places like /usr/local/bin and /usr/local/lib.
