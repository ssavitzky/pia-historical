Tag Handlers for the DPS

[..] [files]

This directory contains primarily tag-handler classes (implementations of crc.dps.Handler) for use as actor handles. They are loaded dynamically.

Base Classes

AbstractHandler
Abstract base class for handlers.
BasicHandler
Class for ``non-active'' node handlers.
GenericHandler
Class for the handlers of active nodes.
LegacyHandler
Class for handlers of ``legacy'' Actor nodes. Processing is delegated to the corresponding handler in crc.interform.handle.

Handler Classes

Tag handler classes have lowercase names, with periods and hyphens replaced by underscores. If the name of a tag happens to match a Java keyword (for example, if) the string Handler must be suffixed to the name. It is optional otherwise.
