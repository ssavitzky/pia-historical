Tag Handlers for the DPS

[..] [files]

This directory contains primarily tag-handler classes (implementations of crc.dps.Handler) for use as actor handles. They are loaded dynamically.

It may also contain other classes, for example those used to initialize tagsets (_ts suffix).

Base Classes

  • AbstractHandler
  • BasicHandler
  • GenericHandler

Handler Classes

Tag handler classes have lowercase names, with hyphens replaced with underscores. If the name of a tag happens to match a Java keyword (for example, if) the string _handler is suffixed to the name.
