Tag Handlers for the DPS

[..] [files]

This directory contains primarily tag-handler classes (subclasses of crc.dps.Handler) for use as actor handles. They are loaded dynamically.

It may also contain other classes, for example those used to initialize tagsets (_ts suffix).
