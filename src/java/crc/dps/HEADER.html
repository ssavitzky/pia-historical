Annotated Contents

Packages

handle/
The Handle subclasses that implement actions (operations) attached to actors. It turns out that tags is a bad name for a subdirectory, since it's the name used for the ``tag file'' created by the ctags utility. CVS ignores it, and various cleanup and maintenance operations are likely to flag it as deletable.
test/
test cases

Interfaces

Processor
A document processor. Both a consumer and producer of Token objects.

Token
A generalization of Node: either a complete Node, a start tag, or an end tag.
Handler
The object that tells the Processor what to do with a Token, and also tells a Parser how to parse it.

EntityTable
A lookup table for Entity values. Supports context chaining.
Tagset
A lookup table for Handlers. Supports context chaining.

Input
A Token source or stream.
InputStack
A stack of Input objects.
InputStackFrame
An Input that can be linked onto a linked-list InputStack.
Parser
An Input that parses Token's from a character stream.
