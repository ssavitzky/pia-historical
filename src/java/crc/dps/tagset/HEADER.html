Tagsets for the DPS

This directory contains tagset classes (implementations of crc.dps.Tagset) for the Document Processing System. They are loaded dynamically.

BasicTagset

Tagsets (XXML)

Each entry in the following list consists of a tagset definition file and its corresponding documentation:
tagset.ts / tagset.html
The formal and informal documentation for tagset definition files.
basic.ts / basic.html
The primitive operations (i.e. those implemented using Java handlers)
xhtml.ts / xhtml.html
The XXML operations mixed with HTML. (XHML)
xxml.ts / xxml.html
eXtended XML -- the standard set of document processing operations, suitable for inclusion in XML documents.
tsdoc.ts / tsdoc.html
The tagset used for documenting tagsets, by converting them to HTML.
tsdtd.ts / tsdtd.html
The tagset used for converting tagset definition files to DTD's.
