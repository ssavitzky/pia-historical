DPS Active Node classes

[..] [files]

This directory contains classes that are used in the DPS for representing active documents (parse trees).

Interfaces

Interfaces in this package have names that start with ``Active''. In general, they are also implementations of crc.dps.Active and its extension ActiveNode.
ActiveNode

Classes

Classes in this packages have names that start with ``ParseTree'' if they are implementations of ActiveNode. The names of auxiliary classes are less consistent.
