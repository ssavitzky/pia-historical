SGML objects

[..] [files]

This directory contains the crc.sgml package, which provides interfaces and implementations for SGML markup languages and applications.

Also see the InterForm Interpretor.

Annotated Contents

handle/
The Handle subclasses that implement actions (operations) attached to actors.
test/
test cases

Attrs.java
The interface for associations of String attribute names with SGML values.
Items.java
The interface for indexed arrays of SGML items.
SGML.java
The interface for generic SGML objects.

AttrTable.java
Attribute-value pairs where the value is SGML.
Token.java (from IT.pm)
Interform Token. The representation for an HTML element.
Tokens.java (from DS/List.pm)
Lists of Interform Tokens. Used as the content of a Token.
