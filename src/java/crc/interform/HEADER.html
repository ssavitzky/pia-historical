InterForm Interpretor and Actors

This directory contains the all-new, all-singing, all-dancing, resolver-and-actor-based InterForm Interpretor.

Also see the PERL version.

Annotated Contents

handle/
The Handle subclasses that implement actions (operations) attached to actors.
test/
test cases

Actor.java (from IA.pm)
InterForm Actors, which are similar to Agents except that they operate on SGML syntactic elements instead of on HTTP transactions.
Input.java (from II.pm)
the InterForm Interpretor's input stack frame.
Interp.java (from II.pm)
the InterForm Interpretor. It combines an HTML parser with a simplified resolver for applying ``Interform Agents'' to elements.
Parser.java (from Parser.pm)
Actually just a scanner--it takes a file or string and produces a stream of Tokens.
Tagset.java (from Tagset.pm)
A (usually named) collection of Actors.
