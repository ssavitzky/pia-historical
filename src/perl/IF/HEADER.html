IF -- InterForm processing

This directory contains the all-new, all-singing, all-dancing, resolver-and-agent-based InterForm code. Note that everything here is in the form of .pm (PERL Module) files. You use the Interform Interpretor with a statement like: 
 use IF::II

Annotated Contents

Actors.pm
Definitions and handlers for the various Actors in the standard tagset.
IA.pm
Interform Agents. These operate similarly to PIA agents, except they operate on SGML syntactic elements instead of on HTTP transactions.
II.pm
the Interform Interpretor. It combines an HTML parser with a simplified resolver for applying ``Interform Agents'' to elements.
IT.pm
Interform Tokens. The replacement for HTML::Element. Unlike the latter, it uses no ``parent'' links, so reference-counting works properly.
Parser.pm
Actually just a scanner--it takes a file or string and produces a stream of Tokens.
Run.pm
Simplified interfaces for running the Interpretor.
Semantics.pm
Utility routines for manipulating Tokens, used by Actors.
Tagset.pm
A (usually named) collection of Actors.
