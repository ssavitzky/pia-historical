PIA -- PERL Implementation of Agencies

This directory will eventually contain PERL modules for the Agency. Note that everything here is intended to be in the form of .pm (PERL Module) files. The idea is that you use a statement like: 
 use PIA::Generic
 ...
 $agent = PIA::Generic->new;
instead of the old 
 require pia_generic_agent.pl
 ...
 $agent = PIA_AGENT::GENERIC->new;
