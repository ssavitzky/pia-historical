PIA -- PERL Implementation of Agencies

This directory contains PERL modules for the Agency. Also see:
  • ../IF/ for InterForm-related modules
  • ../DS/ for the fundamental data structures
Note that everything here is in the form of .pm (PERL Module) files. The idea is that you use a statement like: 
 use PIA::Agent
 ...
 $agent = PIA::Agent->new
instead of the old 
 require pia_agent.pl
 ...
 $agent = PIA_AGENT->new;

Annotated Contents

Agent/
Directory for subclasses of PIA::Agent, and for other classes that belong in the PIA::Agent namespace (e.g. PIA::Agent::Machine).
