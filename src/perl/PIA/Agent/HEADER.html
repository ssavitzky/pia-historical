PIA::Agent::*

This directory contains PERL modules for subclasses of PIA::Agent, as well as other modules that belong in the PIA::Agent namespace. By convention, subclasses of PIA::Agent have lowercase names. See also:
  • ../ for other PIA-related modules
  • ../Agent.pm for PIA::Agent itself.

Annotated Contents

Machine.pm
PIA::Agent::Machine -- the target for requests directed at an Agent.
Options.pm
PIA::Agent::Options -- cache for values (usually the contents of files) computed from options.
