This is the top-level directory for source code generated by members of the PIA Project at CRC. At the moment it is not clear what will be going in here, since most of our efforts will be centered on Agents and Agencies, but I can envision a sizeable stock of shared utilities being developed. For example, the REST slide-show CGI and the WebFax software will probably end up here.

Annotated Contents

Agents/
InterForms for agents that are either experimental or used primarily for software development. Other agents are in ../Agents/
app/
Separate applications, intended to be installed in /usr/local/bin/ or /usr/bin/X11/. Utilities.
cgi/
CGI scripts and programs, meant to be installed in .../cgi-bin on some server.
lib/
Libraries intended to be installed in /usr/local/lib or one of its subdirectories. In particular, lib/perl/ contains the PERL code libraries for the only agency that currently exists, the one in lib/perl/pia.pl.

Typical Make Targets

make public
Install a package or ``installation kit'' in .../pia/pub/.
make install
Install executables in /usr/local/bin/, libraries in /usr/local/lib/, and man pages in /usr/local/man/.
