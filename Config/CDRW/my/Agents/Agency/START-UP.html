Prototype Agency Start-up File

Any links found in this file will be requested by the agency at start-up. Most useful for doing things like initializing agents and the like. Note that the agency currently makes no attempt to save state across sessions. Customize with the commands: 
  mkdir ~/.pia ~/.pia/Agents ~/.pia/Agents/Agency
  cp pia/Agents/Agency/START-UP.html ~/.pia/Agents/Agency/START-UP.html
