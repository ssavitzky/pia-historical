MB3 Configuration steps

  1. Install a java runtime (for example jdk1.2)
  2. Install the PIA (unpack pia_bin.tgz)
  3. If on a windows platform, make sure that police size of the environment for DOS programs is sufficiently large (1024 -- see properties of PIADIR/bin/pia.bat )
  4. Copy the contents of the my subdirectory to the USRDIR (normally ~/.pia) (this step can be skipped for the MB3 distribution which already has the configuration files in the proper place)
  5. Start the pia by clicking on PIADIR/bin/pia.bat (you may want to put this in the windows startup folder along with a web browser).
  6. (optional) Configure web browsers on the MB3 to use the Agency (localhost:8888 ) as proxy.
  7. (optional) Enable file sharing on the MB3 disk so that users can copy presentations to a local file, enter in the directory name on the MB3 configuration page
