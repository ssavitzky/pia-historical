MB3 Configuration steps (on Windows machine)

  1. Download and install a java runtime (click here for jdk1.2 -- For convenience use default installation directory -- c:\jdk1.2beta2)
  2. Install the PIA (download and unpack pia_bin.tgz -- this file has been tarred and gzipped...we will eventually put a winzip file that uses the install shield to make installation easier on Windows machines.
  3. Copy the contents of the my subdirectory to the USRDIR directory (USRDIR by default is ~/.pia on Unix system, c:\PIAUSERS\nobody on windows. It can be changed in the PIADIR/Config/pia.props file. PIADIR is where the pia_bin.tar files are unpacked.)
  4. Copy the contents of the piajdk.bat to PIADIR/bin/piajdk.bat -- overwriting the existing file which has path names set up for the CD-ROM. Check the properties of this batch file and make sure that the environment memory is set to at least 1024. Now start the Agency using this batch file (you may want to put a shortcut to this in the windows startup folder along with a web browser).
  5. (optional) Configure web browsers on the MB3 to use the Agency (localhost:8888 ) as proxy.
  6. (optional) Enable file sharing on the MB3 disk so that users can copy presentations to a local file, enter in the directory name on the MB3 configuration page
